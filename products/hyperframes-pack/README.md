# HyperFrames Video Template Pack

> Professional video templates for social media, branding, and product launches
> Built with HyperFrames — the AI-native video composition engine

## Contents

This pack includes 5 premium video templates:

1. **Modern Brand Intro** — 5-second animated logo reveal with particle effects
2. **Social Media Promo** — 15-second vertical video for Instagram/TikTok/Reels
3. **Product Showcase** — 30-second product highlight with call-to-action
4. **YouTube Outro** — 10秒 end screen with subscribe button and video suggestions
5. **Announcement Bumper** — 3-second quick bumper for brand announcements

## Requirements

- [HyperFrames CLI](https://hyperframes.ai) (`npm install -g hyperframes`)
- FFmpeg (for rendering)

## Quick Start

```bash
# Navigate to a template
cd templates/brand-intro

# Preview in browser
hyperframes preview

# Render to video
hyperframes render
```

## Customization

Each template is fully customizable:
- Replace placeholder text
- Swap colors via CSS variables
- Add your own media assets
- Adjust timing and transitions
